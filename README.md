# Сбор — веб-приложение (кусок 1: вход и регистрация)

Это настоящее React-приложение, подключённое к Supabase. В этой версии реализован
только вход/регистрация (email и телефон через Telegram) — дальше будем добавлять
карту, события, чат и подписку отдельными кусками.

## Что нужно указать при развёртывании (не в самом коде!)

Две переменные окружения:
- `VITE_SUPABASE_URL` — адрес твоего проекта Supabase
- `VITE_SUPABASE_ANON_KEY` — публичный anon-ключ проекта

Их нужно вписать в настройках хостинга (например, Vercel → Settings → Environment Variables),
а не в файлах — так безопаснее и удобнее менять их без изменения кода.
